ANDROID APP

This is the first project I worked on when I started learning about apps; it includes a lot of basic knowledge.

Target users and key features

This app is designed for people who have difficulty communicating in society. It has some very unique features. There is an admin section where tasks and encouraging messages can be added for each mood. Below the admin section is the user section, where users need to input their mood and click on it. Once clicked, an encouraging message will appear at the top, and a task will show up in the middle. The user can complete the task, or if it seems too difficult, they can skip it.

Technologies Used

The technologies I used are:
- Adndroid Studio
I chose this technology because it allows me to create apps for Android devices and it is very easy to use.
