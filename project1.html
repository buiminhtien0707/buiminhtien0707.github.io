TIME MANAGEMENT AND TASK ASSIGNMENT

A website project for time management and task assignment

Target users and key features

This is a website I aimed at everyone, especially students, to help them more easily manage their time. The website also allows people to assign tasks to others and set deadlines for those tasks.

Technologies Used

The technologies I used are:
- Frontend: React
- Backend: NodeJs
- Database: PostgreSQL
I feel that these tools are very easy to use, and most importantly, there are plenty of tutorials available on YouTube

Languages

The languages I have used are: HTML, CSS, Js, React Js, SQL, JSON.
